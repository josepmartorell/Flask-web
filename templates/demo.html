Leading projects

My philosophy is that when we create a platform for use by our company, we understand the usefulness and the functionality we expect from it in such a way that it is relatively easy to develop it. However, when we design a platform for our client this can be somewhat more difficult, in such circumstances it is necessary to debate and discuss said project in great detail, deducing from it an effective minimum structure that allows us to direct the project with maximum efficiency.

About my skills I am specialized in web application development with Flask using Linux operating system for serialized production. Creation of corporate databases with PostgreSQL and MySQL. Cloud computing with AWS to host and launch projects using basic services like S3, EC2 Route 53 and others more to obtain scalability. Auxiliary skills with beautifulsoup, scrapy with Selenium and Splash javascript rendering services. Python automated spreadsheets for office tasks.

My principal task consist of flaskerize smart templates to include useful functionalities and customize them to perform web products. Another one is to automation tasks and data extraction with web scraping techniques.
In summary, all those technological areas of development that deserve my attention are in which I have invested countless hours to progressively make them scalable.
